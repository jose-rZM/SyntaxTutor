void LL1\+Parser\+::\+Compute\+Follow\+Sets (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Computes the FOLLOW sets for all non-\/terminal symbols in the grammar. The FOLLOW set of a non-\/terminal symbol A contains all terminal symbols that can appear immediately after A in any sentential form derived from the grammar\textquotesingle{}s start symbol. Additionally, if A can be the last symbol in a derivation, the end-\/of-\/input marker ({\ttfamily \textbackslash{}\$}) is included in its FOLLOW set. This function computes the FOLLOW sets using the following rules\+: 


\begin{DoxyEnumerate}
\item Initialize FOLLOW(\+S) = \{ \$ \}, where S is the start symbol.
\item For each production rule of the form  A \rightarrow \alpha B \beta\+:
\begin{DoxyItemize}
\item Add FIRST(\beta) \setminus \{\epsilon\} to  FOLLOW(B).
\item If \epsilon \in FIRST(\beta), add FOLLOW(A) to FOLLOW(B).
\end{DoxyItemize}
\item Repeat step 2 until no changes occur in any FOLLOW set. The computed FOLLOW sets are cached in the {\ttfamily follow\+\_\+sets\+\_\+} member variable for later use by the parser. \begin{DoxyNote}{Note}
This function assumes that the FIRST sets for all symbols have already been computed and are available in the {\ttfamily first\+\_\+sets\+\_\+} member variable. 
\end{DoxyNote}

bool LL1\+Parser\+::\+Create\+LL1\+Table (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Creates the LL(1) parsing table for the grammar. 

This function constructs the LL(1) parsing table by iterating over each production in the grammar and determining the appropriate cells for each non-\/terminal and director symbol (prediction symbol) combination. If the grammar is LL(1) compatible, each cell will contain at most one production, indicating no conflicts. If conflicts are found, the function will return {\ttfamily false}, signaling that the grammar is not LL(1).


\begin{DoxyItemize}
\item For each production rule A \rightarrow \alpha, the function calculates the prediction symbols using the {\ttfamily Prediction\+Symbols} function.
\item It then fills the parsing table at the cell corresponding to the non-\/terminal {\ttfamily A} and each prediction symbol in the set.
\item If a cell already contains a production, this indicates a conflict, meaning the grammar is not LL(1).
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
{\ttfamily true} if the table is created successfully, indicating the grammar is LL(1) compatible; {\ttfamily false} if any conflicts are detected, showing that the grammar does not meet LL(1) requirements. 
\end{DoxyReturn}

Calculates the FIRST set for a given production rule in a grammar. 

The FIRST set of a production rule contains all terminal symbols that can appear at the beginning of any string derived from that rule. If the rule can derive the empty string (epsilon), epsilon is included in the FIRST set.

This function computes the FIRST set by examining each symbol in the production rule\+:
\begin{DoxyItemize}
\item If a terminal symbol is encountered, it is added directly to the FIRST set, as it is the starting symbol of some derivation.
\item If a non-\/terminal symbol is encountered, its FIRST set is recursively computed and added to the result, excluding epsilon unless it is followed by another symbol that could also lead to epsilon.
\item If the entire rule could derive epsilon (i.\+e., each symbol in the rule can derive epsilon), then epsilon is added to the FIRST set.
\end{DoxyItemize}

\begin{DoxyItemize}
\item The prediction symbols for a rule determine the set of input symbols that can trigger this rule in the parsing table. This function calculates the prediction symbols based on the FIRST set of the consequent and, if epsilon (the empty symbol) is in the FIRST set, also includes the FOLLOW set of the antecedent.

\end{DoxyItemize}

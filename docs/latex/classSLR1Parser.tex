\doxysubsection{Detailed Description}
Implements an SLR(1) parser for context-\/free grammars. 

This class builds an SLR(1) parsing table and LR(0) automaton from a given grammar. It provides methods for computing closure sets, GOTO transitions, constructing states, and performing syntax analysis using the generated table.

Represents the action table for the SLR(1) parser. 

The action table is a map that associates each state and input symbol with a specific action ({\ttfamily Shift}, {\ttfamily Reduce}, {\ttfamily Accept}, or {\ttfamily Empty}). It is used to determine the parser\textquotesingle{}s behavior during the parsing process.

The table is structured as\+:
\begin{DoxyItemize}
\item Outer map\+: Keys are state IDs (unsigned int).
\item Inner map\+: Keys are input symbols (std\+::string), and values are {\ttfamily \doxylink{structSLR1Parser_1_1s__action}{s\+\_\+action}} structs representing the action to take.

Represents the transition table for the SLR(1) parser. 

The transition table is a map that associates each state and symbol with the next state to transition to. It is used to guide the parser\textquotesingle{}s state transitions during the parsing process.

Computes the FOLLOW sets for all non-\/terminal symbols in the grammar. The FOLLOW set of a non-\/terminal symbol A contains all terminal symbols that can appear immediately after A in any sentential form derived from the grammar\textquotesingle{}s start symbol. Additionally, if A can be the last symbol in a derivation, the end-\/of-\/input marker ({\ttfamily \textbackslash{}\$}) is included in its FOLLOW set. This function computes the FOLLOW sets using the following rules\+: 


\begin{DoxyEnumerate}
\item Initialize FOLLOW(\+S) = \{ $ \$ $ \}, where S is the start symbol.
\item For each production rule of the form  $ A \rightarrow \alpha B \beta
$\+:
\begin{DoxyItemize}
\item Add $ FIRST(\beta) \setminus \{\epsilon\} $ to  $ FOLLOW(B)
$.
\item If $ \epsilon \in FIRST(\beta) $, add $ FOLLOW(A) $ to $ FOLLOW(B) $.
\end{DoxyItemize}
\item Repeat step 2 until no changes occur in any FOLLOW set. The computed FOLLOW sets are cached in the {\ttfamily \doxylink{classSLR1Parser_a1412b54cd36ecf31bf48ee0e239d9671}{follow\+\_\+sets\+\_\+}} member variable for later use by the parser. \begin{DoxyNote}{Note}
This function assumes that the FIRST sets for all symbols have already been computed and are available in the {\ttfamily \doxylink{classSLR1Parser_a4b9b4f88bfef3c3e77f791446f3d820c}{first\+\_\+sets\+\_\+}} member variable. 
\end{DoxyNote}

Calculates the FIRST set for a given production rule in a grammar. 

The FIRST set of a production rule contains all terminal symbols that can appear at the beginning of any string derived from that rule. If the rule can derive the empty string (epsilon), epsilon is included in the FIRST set.

This function computes the FIRST set by examining each symbol in the production rule\+:
\begin{DoxyItemize}
\item If a terminal symbol is encountered, it is added directly to the FIRST set, as it is the starting symbol of some derivation.
\item If a non-\/terminal symbol is encountered, its FIRST set is recursively computed and added to the result, excluding epsilon unless it is followed by another symbol that could also lead to epsilon.
\item If the entire rule could derive epsilon (i.\+e., each symbol in the rule can derive epsilon), then epsilon is added to the FIRST set.
\end{DoxyItemize}
